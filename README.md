## MySQL Design
1. Ratings Table
- `shirt_id`
- `bottom_id`
- `rating`

2. Shirts Table
- `shirt_id`
- `shirt_name`
- `shirt_img` : amazons3 link to shirt image

3. Bottoms Table
- `bottom_id`
- `bottom_name`
- `bottom_img` : : amazons3 link to shirt image
